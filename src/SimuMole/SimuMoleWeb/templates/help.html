PDB file

Protein Data Bank (PDB) file is a textual file that describing the three-dimensional structures of the molecules by storing ATOM records (which represents atomic coordinates, occupancy, temperature factor and more) for each atom in the molecule.
You can find PDB files describing a variety of molecules here and you can also generate your own PDB file (note that this includes learning the mechanics of a PDB file or use some external tool).


DCD file

DCD trajectory file is a binary file that contains the atomic coordinates of the atoms for every time step of the simulation.
You can download the DCD file generated by our Simulation and use it later in our Simulation or in other programs (like PyMOL) as well.

How to view a Simulation using existing PDB and DCD files?

1. Click on the Upload PDB & DCD tab.
2. Click on Choose File, navigate to your PDB / DCD file and open it. Do it for both PDB and DCD files, and make sure they are corresponding.
3. Click on Upload.


How to view your proteins in the PyMOL program?

1. Make sure you have PyMOL installed on your computer. If not, you can download it from here and then follow the instructions to install it.
2. After you open PyMOL, click the file tab and then click the open option.
3. In the open window, navigate to your PDB file and open it. You will see the proteins that the PDB file represent.


How to view your simulation in the PyMOL program?

1. Do the steps above to open the PDB file.
2. Repeat 1. with the DCD file to open the DCD file (make sure that the PDB file and the DCD file are fit). In the popped-up window, click on load.
3. click on the play button in the right bottom corner.


Note that our capability of setting the relative position between molecules is limited, so we encourage you to upload 1 PDB file that contains the merged molecules. You can do that using the PyMOL program.
